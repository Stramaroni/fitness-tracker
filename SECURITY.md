# Sicurezza e privacy

Questa app e statica: non invia dati a un server e salva tutto nel browser del dispositivo con `localStorage`.

## Raccomandazioni

- Pubblica solo su HTTPS, per esempio GitHub Pages.
- Non caricare online file di backup JSON o CSV: contengono dati personali.
- Proteggi l'iPhone con codice, Face ID o Touch ID.
- Fai backup JSON periodici e conservali in un posto privato.
- Non importare backup ricevuti da altre persone.

## Limiti

- `localStorage` non e cifrato dall'app.
- Se cancelli dati del sito da Safari, perdi i dati non esportati.
- Chi ha accesso sbloccato al telefono puo leggere i dati nell'app.
